Meus Projetos em Destaque

CityMove: Sistema de Mobilidade Urbana Inteligente

Solução de rastreamento em tempo real, roteirização e gestão de frotas.

Desenvolvimento de um backend escalável com **Node.js** para lidar com *WebSockets* e dados de geolocalização em tempo real. Implementação de algoritmos de otimização de rotas e frontend (React Native ou Web) para usuários e gestores.

Demo GitHub

FintechControl: Sistema de Gestão Financeira Segura

Plataforma para controle de fluxo de caixa, orçamentos e relatórios financeiros.

Estrutura de dados **PostgreSQL** focada em integridade e segurança. Backend com **Python/Django** para processamento de transações e geração de relatórios detalhados. Foco em criptografia e autenticação robusta.

Demo GitHub

RouteMaster: Otimização Logística

Ferramenta que soluciona o Problema do Caixeiro Viajante (TSP) para rotas de entrega.

Implementação de lógica de roteirização algorítmica em **Python**. Frontend interativo em **Vue.js** com visualização de mapa para definição de pontos e exibição da rota otimizada. Foco em performance de cálculo.

Demo GitHub

BetWave: Plataforma de Apostas Esportivas

Sistema de alto tráfego para gestão e atualização de Odds em tempo real.

Motor de regras de negócio em **Java Spring Boot** para alta concorrência. Utilização de **Redis** para caching de odds em tempo real. Módulos complexos de processamento de transações e liquidação de apostas.

Demo GitHub
